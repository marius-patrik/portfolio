# Liqid Docs

Official documentation website for the Liqid component library.

## Stack

- **Framework**: React 19
- **Build Tool**: Rsbuild
- **Styling**: Tailwind CSS v4

## Commands

| Command           | Description              |
| ----------------- | ------------------------ |
| `npm run dev`     | Start development server |
| `npm run build`   | Build for production     |
| `npm run preview` | Preview production build |
